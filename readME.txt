1. Maak een nieuwe (EMPTY!) cmake project
File -> New -> C/C++ project -> Empty or Existing Cmake project
Geef het de volgende naam voor je project (om het snel compileerbaar te kunnen krijgen): OpenCVBeroepsProductKethanPonnuthurai
2. Extract de files van van de zip en zet ze in het nieuwe project.
3. Build het project.
4. Run de applicatie (staat default op interactive modus).
5. Als je het op de batchmodus wilt runnen, dan moet je een txt file argument doorgeven aan je executable.
